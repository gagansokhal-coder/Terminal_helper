# Quick Install

```bash
curl -fsSL https://raw.githubusercontent.com/gagansokhal-coder/Terminal_helper/main/scripts/install-online.sh | bash
```
*This bootstrap script detects your architecture, installs the latest pre-built binaries, and configures your shell hooks automatically.*

---

# Verify Installation

```bash
ggnmem version
ggnmem doctor
```

---

# Start Daemon

```bash
ggnmem start
ggnmem status
```

---

# Import Existing History

```bash
ggnmem import auto
```

---

# First Searches

```bash
ggnmem search docker
ggnmem search postgres
ggnmem semantic "git branch"
```

---

# Natural Language Examples

```bash
ggnmem ask "show running containers"
ggnmem ask "create postgres backup"
ggnmem ask "find cargo commands"
```

# Self Update

ggnmem self-update
